Click the button to track how many times you've clicked the button.

The results will display both localStorage and sessionStorage. They'll appear the same, but if you click the Home button and then come back to this topic without hitting the reset button, the localStorage count will continue where it left off and the sessionStorage will continue where it left off as long as the browser hasn't been closed. If you close the browser, the sessionStorage will reset, but the localStorage will not. Try it out!

It works the same thing when you store information. If you were to store the information, the localStorage and sessionStorage would return the same information. However, if you were to close the browser without clicking Reset and immediately click Show, the localStorage will still be the same yet the sessionStorage would display nothing.
